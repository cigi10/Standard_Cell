# Standard_Cell

Custom-designed standard cells (INV, NAND, NOR) in 180nm technology,  
characterized and used to synthesize a half adder.

## Cells Included
- Inverter
- 2-input NAND
- 2-input NOR

## Flow
Schematic → Layout → Characterization → Half Adder Synthesis
